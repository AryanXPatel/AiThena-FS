üöÄ Quick Start Guide

1. Clone & Install

git clone <repository-url>
cd AiThena-FS
npm install  # In each service directory

2. Environment Setup

# Create .env files in each backend
GEMINI_API_KEY=your_google_gemini_api_key
QDRANT_URL=http://localhost:6333
PORT=4001/4101/4201

3. Start Services

# Start Qdrant (DocIntel-Backend directory)
docker-compose up -d

# Start each backend service
npm run dev

# Start frontend
cd AiThena-FrontEnd && npm run dev

4. Access Application

  • Frontend: http://localhost:3000
  • AiThena API: http://localhost:4001
  • DocIntel API: http://localhost:4101
  • FlashcardStudio API: http://localhost:4201
